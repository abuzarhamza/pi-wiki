# Step-by-Step Guide

1. Install Node.js using your package manager (e.g., 'brew install node').
2. Install Pi globally by running 'npm install -g @mariozechner/pi-coding-agent'.
3. Set the GEMINI_API_KEY environment variable using 'export GEMINI_API_KEY=your_key'.
4. Select the desired model by using the '/model' command in the Pi interface.
5. Initialize the git-wiki skill by running '/skill:git-wiki init'.
6. Sync documentation with code by running '/skill:git-wiki sync'.
